# NEREIDS Test Data

## LANL-ORNL_example.tif

Synthetic neutron transmission imaging data generated by LANL, containing two
overlapping institution logos (ORNL oak leaf and LANL atom/orbital symbol),
each composed of a different isotope.

**Source**: PLEIADES test data repository (code.ornl.gov), included as a git submodule at `tests/data/pleiades_data/`

### Setup

```bash
# Clone with submodules
git clone --recurse-submodules git@github.com:ornlneutronimaging/NEREIDS.git

# Or initialize after cloning
git submodule init
git submodule update

# Fetch LFS data (the TIFF is stored with Git LFS on code.ornl.gov)
cd tests/data/pleiades_data && git lfs pull
```

### Properties

| Property | Value |
|----------|-------|
| Format | Multi-page TIFF (500 pages), float32 |
| Spatial dimensions | 256 x 256 pixels |
| Energy bins | 500 |
| Energy range | 1 -- 50 eV |
| Energy spacing | Linear (`np.linspace(1.0, 50.0, 500)`) |
| Temperature | 293.6 K |

### Isotope Composition (Ground Truth)

| Logo | Isotope | Areal density (g/cm^2) | Number density (atoms/barn) |
|------|---------|------------------------|-----------------------------|
| ORNL (oak leaf) | U-235 | 0.4 | 0.001025 |
| LANL (atom/orbital) | Pu-241 | 0.2 | 0.0004998 |
