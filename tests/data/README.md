# NEREIDS Test Data

## venus_hf_120min_sample.h5 / venus_hf_open_beam.h5

Real measured neutron transmission imaging data from the VENUS beamline at
SNS, ORNL. A 256×256 spatial crop of a 120-minute Hf foil resonance run
plus the matching open-beam, in NeXus/rustpix HDF5 format consumable by
the NEREIDS GUI loader and `nereids.load_nexus_histogram(...)`.

**Source**: PLEIADES test data repository (code.ornl.gov), in the
`tests/data/pleiades_data/` git submodule. After `git submodule update`,
files appear at:

- `tests/data/pleiades_data/venus_hf_120min_sample.h5` (213 MB, Git LFS)
- `tests/data/pleiades_data/venus_hf_open_beam.h5` (147 MB, Git LFS)

### Properties

| Property | Value |
|----------|-------|
| Format | HDF5 (rustpix NeXus schema), gzip=9 + shuffle, Git LFS |
| Spatial dimensions | 256 × 256 (centered crop of 512×512) |
| TOF bins | 4367 |
| Energy range | 6.67 eV – 2.6 MeV (full); 7–200 eV (production analysis) |
| Flight path | 25.0 m |
| Sample dwell time | 120 min |
| Sample foil | Hf natural |

### Schema

```
/entry                                  attrs: flight_path_m=25.0,
                                                tof_offset_ns=0.0, title
/entry/histogram/counts                 u64 4D [rot_angle=1, y, x, tof]
                                        ASCENDING TOF
/entry/histogram/time_of_flight         f64 1D, ns, ASCENDING (length=tof+1)
```

No proton-charge metadata is stored (not part of the rustpix schema);
the original VENUS run used `c = Q_s / Q_ob ≈ 5.98`. Without it,
callers can recover `c` empirically by anchoring the high-energy
transmission baseline to T → 1.

### Usage

```python
import numpy as np
import nereids

sample = nereids.load_nexus_histogram(
    "tests/data/pleiades_data/venus_hf_120min_sample.h5"
)
ob = nereids.load_nexus_histogram(
    "tests/data/pleiades_data/venus_hf_open_beam.h5"
)
# sample.counts.shape == (4367, 256, 256)  (tof, y, x), ASCENDING TOF
# sample.tof_edges_us.shape == (4368,)     ASCENDING TOF μs
```

### Regenerating

The fixtures are produced from the private 38 GB cube
`.research/spatial-regularization/data/counts/resonance_data_2cm.h5`
by `scripts/fixtures/extract_venus_hf_nexus.py`. The source is not
redistributable. To regenerate locally:

```bash
pixi run python scripts/fixtures/extract_venus_hf_nexus.py
```

## LANL-ORNL_example.tif

Synthetic neutron transmission imaging data generated by LANL, containing two
overlapping institution logos (ORNL oak leaf and LANL atom/orbital symbol),
each composed of a different isotope.

**Source**: PLEIADES test data repository (code.ornl.gov), included as a git submodule at `tests/data/pleiades_data/`

### Setup

```bash
# Clone with submodules
git clone --recurse-submodules git@github.com:ornlneutronimaging/NEREIDS.git

# Or initialize after cloning
git submodule init
git submodule update

# Fetch LFS data (the TIFF is stored with Git LFS on code.ornl.gov)
cd tests/data/pleiades_data && git lfs pull
```

### Properties

| Property | Value |
|----------|-------|
| Format | Multi-page TIFF (500 pages), float32 |
| Spatial dimensions | 256 x 256 pixels |
| Energy bins | 500 |
| Energy range | 1 -- 50 eV |
| Energy spacing | Linear (`np.linspace(1.0, 50.0, 500)`) |
| Temperature | 293.6 K |

### Isotope Composition (Ground Truth)

| Logo | Isotope | Areal density (g/cm^2) | Number density (atoms/barn) |
|------|---------|------------------------|-----------------------------|
| ORNL (oak leaf) | U-235 | 0.4 | 0.001025 |
| LANL (atom/orbital) | Pu-241 | 0.2 | 0.0004998 |
