# Requirements (Draft)

## Functional
- Compute resonance transmission with declared isotopes.
- Support user-defined resolution functions.
- Self-indication and cylindrical sample transmission corrections.
- Unresolved resonance handling and isotope discovery workflows.
- NeXus/HDF5 compliant I/O.
- Python bindings for analysis workflows.

## Non-functional
- Reproducible, testable physics kernels.
- Clear metadata and provenance in outputs.
- Extensible architecture with stable interfaces.
- Performance suitable for imaging pipelines.
