# Roadmap (Draft)

## Phase 0: Planning and references
- Capture SAMMY reference map and match to required features.
- Define data models and a minimal forward model API.

## Phase 1: Forward model MVP
- Transmission calculation for declared isotopes.
- User-defined resolution functions.
- Self-indication and cylindrical corrections.
- NeXus/HDF5 I/O and Python bindings.

## Phase 2: Inference and diagnostics
- Abundance estimation workflows and uncertainty reporting.
- Unresolved resonance handling and isotope discovery support.

## Phase 3: Productization
- CLI and GUI workflows for beamline users.
- Validation datasets and performance benchmarking.
